##	 Documtent

自定义弹出视图，视图中的文字可以动态计算其高度。

## Usage

	#import "XXAlertView.h"
	
	[[XXAlertView sharedAlertView] showAlertViewWithTextString:@"你好牛逼啊！你好牛逼啊！你好牛逼啊！你好牛逼啊！你好牛逼啊！"];
